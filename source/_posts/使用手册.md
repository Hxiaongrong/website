你可以执行下列命令来创建一篇新文章或者新的页面。
```
$ hexo new [layout] <title>
```

你可以执行下列操作来删除页面。
```
到目录/source/_posts下删除相应的文章
$ hexo g
$ hexo d
```

你可以执行下列操作来更新页面至github。
```
hexo clean && hexo deploy
```

可参考链接：
[https://hexo.io/zh-cn/docs/](https://hexo.io/zh-cn/docs/ "hexo")
